# 学んだこと
`format.html { redirect_to cart_path(@cart), notice: '商品がカートに追加されました。' }`
ここの部分でカートに追加後の遷移先を選択する。
`format.html { render plain: "商品をカートに追加しました。" }`
これだとカートページには遷移せず、商品ページのままになる。
