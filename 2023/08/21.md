# 取り組んだ課題
Docker

## 学んだこと
* 新しいパッケージリストを取得
```
apt-get update
```

* パッケージインストール
```
apt-get install <package>
```

* RUNが増やさないために、`&&`で繋げて書く。
* installも複数で書けるが見やすさのためにバックスラッシュで改行＆アルファベット順に並べる
* dockerファイルをメンテナンスするときはRUNを複数作成、最終的に&&を使ってまとめる
* 基本は`CMD ["/bin/bash"]`をつける
* RUNはレイヤーを作る、CMDは作らない
