# 学んだこと
```
SELECT その行の内容を取得
UPDATE その行の内容を書き換える
DELETE その行を削除

コメントの記述 `-- コメント`

AS 別名で保存

SELECT データの検索
UPDATE データの更新
DELETE データの削除
INSERT データの追加

※WHEREのない、UPDATEとDELETEは全件対象になる

* 4大命令の分類
検索系：SELECT
更新系：UPDATE、DELETE、INSERT

既存系：SELECT、UPDATE、DELETE
新規系：INSERT
```
